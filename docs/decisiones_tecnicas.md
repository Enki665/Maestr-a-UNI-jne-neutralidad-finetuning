\# Decisiones técnicas iniciales



\## Enfoque principal



El proyecto se orienta al fine-tuning de un modelo de lenguaje natural para la evaluación asistida de proyectos de resolución en materia de neutralidad electoral.



\## Enfoque complementario



Se incorporará recuperación documental para brindar trazabilidad normativa, citas verificables y sustento jurídico a las respuestas generadas.
